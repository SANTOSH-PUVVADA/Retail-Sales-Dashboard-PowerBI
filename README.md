# Retail Sales Dashboard | Power BI

## Project Overview
This project is an interactive Retail Sales Dashboard developed in Power BI to analyze sales performance, profitability, customer behavior, and product trends.

## Features
- Data Cleaning using Power Query
- Data Modeling
- DAX Measures & KPIs
- Interactive Filters & Slicers
- Drill-down Analysis

## Dashboard Includes
- Total Sales by Month Year
- Total Profit by Category
- Total Sales by Region
- Total Sales by Customer Segment
- Average Rating by Customer Segment
- Total Sales by Age Group
- Profit Margin by Category
- Total Sales by Category and Supplier
- Total Sales by Product
- Total Sales by Payment Method
- Profit Margin by Brand

## Skills Used
- Power BI
- DAX
- Power Query
- Data Cleaning
- Data Modeling
- Data Visualization
